# Selenium Cura Automation

##  Project Description

This project contains automated test cases for the Cura Healthcare demo website using Selenium WebDriver with Java and TestNG.

##  Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven

##  Test Scenarios

* Login with valid credentials
* Appointment booking
* Log out

##  How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run tests using TestNG or:

```
mvn test
```

##  Project Structure

* `base` – WebDriver setup
* `pages` – Page Object Model classes
* `tests` – Test cases

##  Application Under Test

https://katalon-demo-cura.herokuapp.com/
