# Selenium Cura Automation

##  Project Description

This project contains automated test cases for the Cura Healthcare demo website using Selenium WebDriver with Java and TestNG.

##  Technologies Used

* Java
* Selenium WebDriver
* TestNG
* Maven

##  Test Scenarios

* Login with valid credentials (Positive testing)
* Login with invalid/empty credentials (Negative testing)
* Appointment booking workflow with different test cases
* Logout functionality

##  How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run tests using TestNG or:

```
mvn test
```

##  Project Structure

* `base` – WebDriver setup
* `pages` – Page Object Model classes
* `tests` – Test cases
* `utils`
